# VisualTkGUI
VisualTkGUI is a tool to make GUI with Python Tkinter Library Visually, i.e., something similar to Visual Studio for WinForms, but for Python and Tkinter.

>[!WARNING]
>This project is under development, First Release (Beta) on 1 August 2026.

# Installation
If you want to just make GUI with this then just install this with installer inside `Releases` folder.

If you want to customize and use it as you want it, then clone the repository and run `main.py` inside `/main` folder using the following command :

```python
python main.py
```
>There are no dependencies required, VisualTkGUI only uses `Python` and `Tkinter` only.

The project structure is quite simple, start exploring from `main.py`.

# Contribution

For Contributing, Start by exploring with `main.py` inside `main/` folder.

### There are mainly Four modules:
- #### VTkGMainApp
    Instanciating main app.
- #### VTkGToolBox 
    This  contains code for toolbox on the left side.
- #### VTkGProperties
    This contains code for properties window on the right side.
- #### VTkGDesignOrText
    - VTkGDesignOrText decides what to show, designer ot text editor.
    - VTkGDesigner - how Designer works.
    - VTkGText - How Text Editor works.

## How to Contribute
